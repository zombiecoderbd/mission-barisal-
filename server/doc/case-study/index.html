📌 প্রকল্প সংক্ষেপ

Mission Barisal v3 একটি জিরো-ডিপেন্ডেন্সি, সিঙ্গেল-ফাইল Node.js সার্ভার যা ৬টি AI Agent-কে অর্কেস্ট্রেট করে। প্রতিটি এজেন্টের আলাদা বাংলা পার্সোনা, এক্সপার্টাইজ এবং মডেল অ্যাসাইনমেন্ট আছে। এজেন্টরা প্রশ্ন নিয়ে ডিবেট করে, ক্রস-ভেরিফাই করে এবং কনসেনসাস আউটপুট তৈরি করে।

✅ দাবি (Claims) — যা কাজ করে

✓ ১. জিরো-ডিপেন্ডেন্সি আর্কিটেকচার

🔴 পরিচিত ইস্যু (Known Issues) — চেক করতে হবে

PENDING Issue 1: Copilot Chat মডেল না দেখা যাওয়া

সমস্যা: Extension-এর LanguageModelChatProvider মডেলগুলো Copilot Chat-এর মডেল পিকার-এ দেখা যায় না বা দেখালেও কাজ করে না।

সম্ভাব্য কারণ:

  • VS Code 1.120.0-এ LanguageModelChatProvider API পরিবর্তিত হতে পারে
  • onDidChangeLanguageModelChatInformation event সঠিকভাবে কাজ নাও করতে পারে
  • Extension activation timing issue — Copilot Chat query করে extension activate হওয়ার আগেই

চেক করতে হবে:

  • vscode.lm.registerLanguageModelChatProvider API কি রানটাইমে exists?
  • Extension activation log চেক করুন (Output → Mission Barisal)
  • VS Code Developer Tools-এ console error চেক করুন

PENDING Issue 2: chatLanguageModels.json Conflict

সমস্যা: ~/.config/Code/User/chatLanguageModels.json-এ Mission Barisal entry থাকলে extension-এর provider-এর সাথে conflict হয়।

বর্তমান অবস্থা: chatLanguageModels.json থেকে Mission Barisal entry সরানো হয়েছে। এখন extension-এর মাধ্যমেই মডেল দেখা যাচ্ছে।

চেক করতে হবে:

  • chatLanguageModels.json ক্লিন — ৪ টি entry, Mission Barisal নেই
  • Extension-এর provider models Copilot Chat-এ visible কিনা

PENDING Issue 3: Diagnostic Command

বর্তমান অবস্থা: "Mission Barisal: Run Diagnostics" কমান্ড যোগ করা হয়েছে। এটি extension-এর প্রোভাইডার রেজিস্ট্রেশন এবং সার্ভার কানেকশন টেস্ট করে।

চেক করতে হবে:

  • কমান্ড প্যালেট থেকে ডায়াগনস্টিক রান করুন
  • আউটপুট চেক করুন — সার্ভার connected? LM API exists?

PENDING Issue 4: VS Code Publisher Verification

সমস্যা: zombie-coder পাবলিশার ভেরিফাইড না হওয়ায় VS Code Mission Barisal-কে "অবিশ্বস্ত" প্রোভাইডার হিসেবে দেখাতে পারে।

চেক করতে হবে:

  • VS Code Marketplace-এ publish করা দরকার কিনা

PENDING Issue 5: Cache_control Artifact

সমস্যা: Terminal এবং tool আউটপুটে {$mid:24,mimeType:"cache_control",data:"ZXBoZW1lcmFs"} নামক আর্টিফ্যাক্ট দেখা যাচ্ছে। এটি VS Code-এর অভ্যন্তরীণ ক্যাশিং সিস্টেমের artifact, ফাইলের কোনো ক্ষতি করেনি।

চেক করতে হবে:

  • VS Code আপডেট করলে সমস্যা যায় কিনা
  • অথবা VS Code রিস্টার্ট করলে ক্লিয়ার হয় কিনা

PENDING Issue 6: Audio File Size

সমস্যা: doc/ ফোল্ডারে MP3 ফাইলগুলো (২-৩ MB) .gitignore-এ *.mp3 excluded থাকায় normal git add-এ যোগ হয় না।

বর্তমান অবস্থা: git add -f ব্যবহার করে force-add করা হয়েছে এবং GitHub-এ push করা সফল। মোট ৭টি MP3 ফাইল (২ MB full version সহ) পুশ করা হয়েছে।

সতর্কতা: পরবর্তী commit-এ MP3 ফাইলগুলো ignore হবে যদি আবার force add না করা হয়।

PENDING Issue 7: Agent Image (JPG/PNG) Reading

সমস্যা: এজেন্টরা JPG, PNG, GIF ইত্যাদি ইমেজ ফরমেট পড়তে পারেনা। শুধুমাত্র টেক্সট ফাইল পড়তে পারে।

কারণ: বর্তমান MCP টুলস শুধুমাত্র টেক্সট ফাইল রিড করে (read_file)। ইমেজ প্রসেসিং এর জন্য আলাদা টুল বা OCR সাপোর্ট লাগবে।

সম্ভাব্য সমাধান:

  • MCP-তে নতুন read_image টুল যোগ করা (base64 এনকোড করে)
  • অথবা vision-capable মডেল ব্যবহার করা (GPT-4 Vision, Gemini Vision)
  • OCR পাইপলাইন যোগ করা (Tesseract ইত্যাদি)
  • লো-প্রায়োরিটি — বর্তমানে টেক্সট-ভিত্তিক কাজই প্রধান ফোকাস

বর্তমান অবস্থা: গবেষণাধীন — টেক্সট এজেন্টদের জন্য ইমেজ রিডিং নেটিভভাবে সম্ভব নয়। Vision মডেল বা OCR টুল লাগবে।
